## Study Strategy & Tips

### For Beginners
1. Start with **Easy** problems in each category
2. Master basic implementations before moving to variations
3. Focus on understanding time/space complexity
4. Practice writing clean, bug-free code

### For Intermediate
1. Focus on **Medium** problems and pattern recognition
2. Learn to identify when to use each technique
3. Practice competitive programming problems
4. Time yourself to improve speed

### For Advanced
1. Tackle **Hard** problems and complex variations
2. Study advanced optimizations and edge cases
3. Participate in contests (Codeforces, AtCoder, etc.)
4. Focus on implementation speed and accuracy

### Key Patterns to Master

#### Binary Search Patterns
- **Classic Binary Search**: Finding target in sorted array
- **Binary Search on Answer**: Optimizing a function
- **Binary Search on Real Numbers**: Continuous search space
- **2D Binary Search**: Matrix and grid problems

#### Two Pointers Patterns
- **Opposite Direction**: Start from ends, move toward center
- **Same Direction**: Both pointers move in same direction
- **Sliding Window**: Maintain window with specific properties
- **Fast & Slow**: Different speeds for cycle detection

#### Sorting Patterns
- **Custom Comparators**: Sorting by complex criteria
- **Merge Techniques**: Combining sorted sequences
- **Counting & Bucket Sort**: Non-comparison sorting
- **Interval Problems**: Sorting by start/end times
